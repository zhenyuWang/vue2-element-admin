# vue2-element-admin

简体中文 | [English](./README.en.md)

#### 介绍
一个基于 `Vue2+Vue Router+Vuex+ElementUI+axios的后台管理系统开发框架`。

#### 技术栈
`Vue2+Vue Router+Vuex+ElementUI+axios`

#### 安装
```shell
yarn install
```
#### 启动服务
```shell
yarn dev
```
